## Quem pode contribuir?

Todo mundo pode ser um contribuidor do repositório, mas para isso só tem de atender algumas regras para conseguir contribuir de maneira simples e organizada.

## Passos para contribuir?

1. Fazer um fork do repositório.
2. Ir para o seu fork, criar uma branch com o seu nome, exemplo: `maria`.
3. Fazer as alterações que pretende nessa branch, adicionando as provas ou materiais.
4. Comitar todas as alterações e abrir um **pull request** no projecto principal.

## Como fazer os commits?

O seu commit deve ser da seguinte forma:

**nome disciplina**:**tipo de conteúdo adicionado**:**informações extras**

Exemplo:

`git commit -m "Fundamentos de Programação 1:Frequências:Adicionei mais 4 provas de frequências"`
